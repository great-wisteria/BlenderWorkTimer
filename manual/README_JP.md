# Blender Work Timer
Blender 4.5.x LTS 向け 作業時間計測アドオン

> **[ 目的 ]**
> 「毎日どれくらいBlenderを触ったか」を可視化して**モチベーションを維持**すること、そして特定の作業（モデリングなど）に**時間をかけすぎていないかを客観的に確認**するためのプラグインです。

![Main Interface](images/main_ui.png)

## インストール方法 (How to Install)

1. 最新版の **[Releases ページ](https://github.com/HasciiCode/BlenderWorkTimer-Release/releases/latest)** から `BlenderWorkTimer.zip` をダウンロードします。（※絶対に解凍しないでください）
2. Blenderを起動し、上部メニューから `編集 (Edit)` > `プリファレンス (Preferences)` を開きます。

![Install Guide](images/install_guide_preference.png)

3. 左側のメニューから `アドオン (Add-ons)` を選択します。
4. 右上の `インストール... (Install...)` ボタンをクリックし、ダウンロードした `BlenderWorkTimer.zip` を選択します。

![Install Guide](images/install_guide_install.png)

5. リストに表示された「System: Blender Work Timer」の横にあるチェックボックスをオンにして有効化します。

![Install Guide](images/install_guide_enable_BWT.png)

---

## 使い方 (Usage)

1. 3Dビューポート上で `N` キーを押し、サイドバーを開きます。
2. **Timer** タブをクリックすると、計測パネルが表示されます。
3. 作業を開始すると自動的に時間がカウントアップされます。
4. ポロモードを使用する場合は、パネル内の「Polo Mode Timer」セクションからStartボタンを押してください。

---

## 主な機能 (Features)

### 1. リアルタイム作業時間トラッキング
Blenderを開いてからの実際の作業時間を自動で計測し、サイドバーに表示します。
- **無操作検知**: 2分間マウスやキーボードの操作がない場合、自動的にタイマーを一時停止し、離席時間を作業時間から除外します。
- **本日と合計時間の表示**: その日の作業時間と、プロジェクト開始からの累計作業時間を一目で確認できます。

![Install Guide](images/work_time.png)

### 2. ポロモード（ポモドーロタイマー）
集中力を維持するための時間管理術（ポモドーロ・テクニック）をBlender内で実践できるモードです。

> **[ ポロモード（ポモドーロ・テクニック）とは？ ]**
> 「短時間の集中作業」と「短い休憩」を交互に繰り返すことで、脳の疲労を防ぎ、深い集中力を長時間維持するためのテクニックです。
> 一般的には「25分の作業＋5分の休憩」が世界的な基本ルールとして有名ですが、ご自身の集中力に合わせて「50分＋10分」など自由にカスタマイズして使うことができます。

- **確実な通知**: 時間が来るとポップアップ画面で通知され、作業と休憩のメリハリをつけることができます。
- **カスタマイズ可能**: 作業と休憩の時間は自由に設定可能です。

![Polo Mode](images/polo_mode.png)

## 注意事項 (Notes)
- このアドオンは `.blend` ファイルを保存した際に、同じフォルダ内に時間データを記録した不可視フォルダを作成します。
- 複数のBlenderを同時に開いた場合でもデータが競合しないように設計されています。
